디지털 정보의 표현

 

비트 : 컴퓨터에서 데이터를 표시할 때 사용하는 최소 단위 2진법 사용 1비트로 표현 할 수 있는 수는 0과 1

바이트 : 비트 8개 묶어서 사용하는 단위 1바이트 = 8비트

 

CPU 처리속도

초기 : 8비트 처리 하는 CPU

이후 : 32비트 CPU

현재 : 64비트 CPU

비트가 많을 수록 처리속도와 명령처리가 빠름

x86 -> 32비트 프로그램(호환성 때문)

 

워드

컴퓨터가 한 번에 처리 할 수 있는 데이터 크기를 나타내는 단위

1워드 = 32비트 한번에 32비트 데이터 처리 32비트 CPU

1워드 = 64비트 한번에 64비트 데이터 처리 64비트 CPU

 

바이트 -> 킬로바이트 -> 메가바이트 -> 기가바이트 -> 테라바이트 -> 페타바이트

1GB = 1000MB

1KB = 1024Byte

 

CPU 속도

클록

컴퓨터에서 일정한 박자를 만들어 내는 장치

컴퓨터 내 모든 부품은 클록이 일정한 간격으로 틱을 만들면 거기에 맞추어 작업

 

헤르츠

CPU 성능을 나타내는 단위

1초에 1번 일정한 박자로 움직이는 것을 1Hz

1초동안 클록틱이 몇 번 발생했는지 나타냄

CPU 속도 3GHz : 1초동안 작업이 약 30억번

메인메모리 속도 1.6GHz : 1초동안 1.6X10^9번 데이터 저장

 

하드디스크 속도(rpm)

디스크 원반이 1분동안 회전하는 수

7,200rpm : 디스크 원반이 1분에 7,200번 회전

수치가 클수록 데이터를 저장하거나 읽는 속도가 다름

 

네트워크 전송량(bps)

네트워크 상에서 1초 동안 보내는 데이터의 양

파일용량 표기 : 바이트(대문자 B)

네트워크 전송량 표기 : 비트(소문자 b)

 

컴퓨터 2진법 사용하는 이유

컴퓨터는 0과 1로 표현하는 2진법 사용

인간은 0부터 9까지 10개로 표현하는 10진법 사용

계산이 빠르고 최소의 투자로 최대의 효과를 얻기 위해서

최초의 컴퓨터가 진공관을 사용했기 때문에(켜기와 끄기만 할 수 있는 진공관 꺼지면 0, 켜지면 1)

 

클라우드 컴퓨팅 서비스

SaaS : 서버안에 있어야 하는 모든 내용 준비해놓고 사용한 만큼 돈만 지불하면 되는 방식(PC방같이)(애플리케이션 데이터 런타임 미들웨어 운영체제 가상화 서버 저장소 네트워크 다 갖춤)

PaaS : 운영체제 설치까지 다 되어 있고 플랫폼까지 제공(어플리케이션 데이터만 준비하면 됨)

laaS : 컴퓨터 인프라 서비스만 준비(런타임, 미들웨어, 운영체제, 가상화, 서버, 저장소, 네트워크만 준비)

온프레미스 : 사무실외 아무것도 없이 맨땅에서 다 하는 것(애플리케이션 데이터 런타임 미들웨어 운영체제 가상화 서버 저장소 네트워크)

 

중요한 데이터 회사에 필요한 클라우드를 프라이빗 클라우드(Private Cloud)

외부에 서비스를 하기 위해서 하는 클라우드를 퍼블릭 클라우드(Public Cloud)
